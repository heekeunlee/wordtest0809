## üöÄ How to Use

### Running Locally
1. Clone or download this repository.
2. Simply open `index.html` in your web browser (Chrome, Safari, Edge, etc.).
3. Choose a Day and start the quiz!

### Deployment (GitHub Pages)
This app is ready to be deployed using GitHub Pages.

1. Go to the **Settings** tab of this repository on GitHub.
2. Click on **Pages** in the left sidebar.
3. Under **Build and deployment**, select **Source** as `Deploy from a branch`.
4. Select `main` (or `master`) branch and `/ (root)` folder.
5. Click **Save**.
6. Wait a minute, and your site will be live!
